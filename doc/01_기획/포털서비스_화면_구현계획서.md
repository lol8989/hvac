# LG 도면 포털 - 포털 서비스단 화면 구현 계획서

> 사용자(견적 담당자)가 도면 업로드부터 산출 엑셀 다운로드까지 수행하는 포털 서비스단의 화면 구성 및 동작을 정의합니다.

---

## 1. 전체 개요

### 1.1 시스템 흐름 요약

```
[사용자]                  [도면 포털]                   [출력]
   │                          │                           │
   ├─ 도면 업로드 ──────────▶ 도면 분석 엔진              │
   │                          │  ├─ 장비 일람표 추출       │
   │                          │  ├─ 시스템_ID 식별        │
   │                          │  ├─ 배관 길이 산출        │
   │                          │  └─ 분기·꺾임 지점 분석    │
   │                          │                           │
   │                          ├─ 마스터 자동 매칭          │
   │                          │  └─ LG 영업팀 등록         │
   │                          │     장비 마스터 적용       │
   │                          │                           │
   ├─ 부자재 단가 입력 ─────▶ 산출 엔진                   │
   │                          │  ├─ 수량 산출 로직 적용    │
   │                          │  ├─ 공수계수 1.1025        │
   │                          │  └─ 자재비·노무비 합산     │
   │                          │                           │
   └─ 산출 엑셀 다운로드 ◀──────────────────────────────▶ 견적 산출 엑셀
```

### 1.2 사용자 권한 구조

| 역할 | 권한 |
|---|---|
| **사용자 (견적 담당자)** | 자신이 작성한 견적·단가만 조회·수정 |
| **관리자** | 전체 견적 조회, 마스터 데이터 관리 |

### 1.3 화면 목록

| # | 화면명 | 주요 기능 |
|---|---|---|
| 1 | 로그인 / 메인 대시보드 | 내 견적 목록, 신규 견적 작성 진입 |
| 2 | 신규 견적 작성 - 도면 업로드 | 도면 파일 업로드, 기본 정보 입력 |
| 3 | 도면 분석 결과 확인 | 추출된 장비 일람표·배관 길이 검토 |
| 4 | 부자재 단가 입력 | 약 50개 부자재 단가 입력 |
| 5 | 산출 결과 미리보기 | 자동 계산된 견적 결과 확인 |
| 6 | 견적 완료 / 다운로드 | 산출 엑셀 다운로드, 견적 저장 |

**주요 요소**
- **입력 방식 3가지 선택지**:

  - 엑셀 양식 업로드: 표준 양식 다운로드 → 작성 → 업로드 (권장)
  - 이전 견적 복사: 사용자 본인의 이전 견적 단가 그대로 가져오기
- **카테고리별 접이식 입력 영역**: 50개 항목을 7개 카테고리로 분류
- **진행 상태 표시**: 입력 완료 항목 수 / 전체
- **공통 변수 영역**: 공수계수, 직종별 노무비 단가 별도 입력
- 임시저장 가능 (이탈 시 데이터 보존)

## 4. 핵심 화면 설계 원칙

### 4.1 단계형 진행 (Stepper UI)
- 견적 작성을 4단계로 분리하여 사용자 인지 부담 최소화
- 상단에 현재 단계 명시, 이전 단계로 자유롭게 이동 가능
- 각 단계는 임시저장되어 중간 이탈에도 데이터 보존

### 4.2 자동화 + 수동 보정 균형
- 도면 분석은 시스템이 자동 수행
- 결과는 사용자가 검토·수정 가능 (장비 변경, HP 보정 등)
- 자동/수동 구분을 시각적으로 명확히 (자동 추출 항목은 ✓ 표시)

### 4.3 부자재 단가 입력 부담 최소화
- 50개 이상 항목의 입력 부담을 3가지 방식으로 완화:
  1. **엑셀 양식 업로드** (가장 권장)
  2. **이전 견적 복사** (반복 작업 효율화)
  3. **직접 입력** (필요 시)

### 4.4 권한별 데이터 격리
- 사용자는 본인이 작성한 견적만 조회·수정
- 관리자는 전체 견적 조회 가능 (관리자 페이지 별도)

## 5. 화면 외 연계 사항

### 5.1 도면 분석 엔진 (백엔드)
- 도면 → 장비 일람표 자동 추출
- 시스템_ID 식별 (실외기 단위)
- 배관 길이 산출 (총 / 주 / 평면 / 수직 층고)
- 분기 지점·꺾임 지점 분석 (엘보·분지관 수량 산출용)

### 5.2 산출 엔진 (백엔드)
- 샘플 엑셀의 수식 로직을 시스템화
- 분지관 차수별 유형 결정: HP > 35 → 114, > 17 → 113, > 9 → 112, 그 외 111
- 냉매량 계산: 배관 구경별 차등 계수 적용 (0.266 / 0.173 / 0.118 / 0.061)
- 실내기 보정계수: HP × 2900 / 실내기수량 비율 기반 (0.32 / 0.48 / 0.64)

### 5.3 마스터 데이터 (관리자 페이지에서 관리)
- 실외기 마스터: 모델명, HP, 공급가, 메타데이터
- 실내기 마스터: 모델명, HP, 공급가, 메타데이터
- LG 영업팀이 등록·갱신
