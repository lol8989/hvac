# public/ — 정적 에셋 (Vite)

이 폴더의 파일은 Vite가 **사이트 루트(`/`)**로 그대로 서빙합니다.
예: `public/sample.dxf` → 앱에서 `fetch('/sample.dxf')` 로 로드.

## 실제 도면 뷰어(DXF) 연동용 파일 놓는 위치

| 파일 | 경로 | 용도 |
|------|------|------|
| 샘플 도면 | `public/sample.dxf` | DXF 벡터 파싱·렌더 대상 |
| (선택) 검출 결과 | `public/detection.json` | 실(방) 폴리곤·라벨·면적 오버레이 정합용 |

`detection.json` 예시(좌표는 **DXF 월드 좌표** 기준):

```json
{
  "units": "mm",
  "rooms": [
    { "id": "AC_001", "name": "거실", "area": 31.89, "cool": 11.2,
      "polygon": [[0,0],[5000,0],[5000,4000],[0,4000]] }
  ]
}
```

넣으신 뒤 "넣었다"고만 알려 주시면 뷰어에 바로 연결합니다.
파일이 없어도 앱은 정상 동작합니다(도면 레이어만 비어 있음).
